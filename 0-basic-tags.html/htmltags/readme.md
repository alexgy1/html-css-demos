## html5 标签

- article

  - 可以平级
  - 可以嵌套
  - 可以内嵌页面

- aside

  - article 里面解释
  - 全局侧边栏

- nav 导航
- section

  - 没有标题的内容不要使用
  - 不要之作为将设置样式的容器，是 div 的工作
  - article nav aside 更符合 不要用 section

- time

  - T 分隔符
  - Z 机器编码 utc 时间
  - +09:00 时差

- pubdate

## 媒体查询

- 外联 link 标签支持 media 属性
- 内嵌 @media 里面属性规则 根据最小和最大宽度来重新调整样式
